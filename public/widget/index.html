Chatbot Widget Demo

About This Demo

This page demonstrates the embeddable chatbot widget. You can interact with the chatbot by clicking the chat icon in the bottom right corner of the screen.

The widget is fully responsive and can be easily integrated into any website.

Widget Features

  • Minimized chat icon that expands on click
  • Responsive design that works on all devices
  • Real-time communication using WebSockets
  • File upload capabilities for PDFs
  • Typing indicators
  • Session persistence across page refreshes

Integration Instructions

To add this chatbot to your website, add the following code snippet just before the closing </body> tag:

<script src="https://your-server-url/widget/js/widget.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    ChatbotWidget.init({
      serverUrl: 'https://your-server-url',
      chatbotName: 'Your Assistant Name',
      primaryColor: '#2196f3'
    });
  });
</script>

Customize the configuration options to match your branding and server URL.

Chatbot Widget Demo

This page demonstrates how the chatbot widget can be embedded in any website.

How to Embed the Widget

Add the following script tag to your website's HTML:

<script src="https://your-domain.com/js/chatbot-widget.js" defer></script>

Customization Options

You can customize the widget by adding a configuration before the script:

<script>
  window.chatbotConfig = {
    primaryColor: '#4CAF50',
    title: 'Property Assistant',
    welcomeMessage: 'Hello! How can I help you today?'
  };
</script>
<script src="https://your-domain.com/js/chatbot-widget.js" defer></script>

Features

  • Responsive design works on all devices
  • Minimizes to a chat icon when not in use
  • File upload/download for PDF documents
  • Real-time messaging with typing indicators
  • Session persistence across page loads

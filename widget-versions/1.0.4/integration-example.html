Welcome to our sample website. This demonstrates how the chatbot widget can be seamlessly integrated into any website.

The widget can be customized to match your brand colors and positioned on either the left or right side of the screen.

Integration Instructions

Adding the chatbot to your website is as simple as adding a single script tag:

<script 
    src="https://your-server-url/widget/js/widget-loader.js" 
    data-server-url="https://your-server-url" 
    data-chatbot-name="Your Assistant Name" 
    data-primary-color="#2196f3" 
    data-position="right"
    data-initial-message="Hello! How can I help you today?">
</script>

This script should be placed just before the closing </body> tag.

Customization Options

  • data-server-url: The URL of your chatbot server
  • data-chatbot-name: The name displayed in the chat header
  • data-primary-color: The main color of the widget (HEX format)
  • data-position: Position of the widget, either "right" or "left"
  • data-initial-message: The first message displayed by the chatbot
